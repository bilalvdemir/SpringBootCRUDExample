# Spring Boot CRUD Example MVC
Sample user service, includes:
 - CREATE, Retrieve, Update, Delete
 - MVC Pattern
 - Web Service
 - Logger
 - Validators and custom validator annotation
 - Exceptions
 - Regex Pattern Matchers
 - Global exception handler
 - MongoDB
 - Message formatter
 
# Used Annotations

 - @Service
 - @RestController
 - @RequestMapping
 - @ResponseBody
 - @GetMapping
 - @PostMapping
 - @PutMapping
 - @DeleteMapping
 - @ControllerAdvice
 - @ExceptionHandler
 - @Id
 - @NotNull
 - @Email
 - @Pattern
 - @Documented
 - @Constraint
 - @Target
 - @Retention
 - Custom Validator Annotation
